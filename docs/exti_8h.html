This function initializes the exti interrupt and nvic interrupts will be received from gpio_port with the pin matching the number of the exti.

Parameters
extithe external interrupt peripheral linked to the gpio pin (number must match !)
gpio_portthe port on which the limit switch will be plugged
interrupt_numberthe interrupt number in the NVIC table
trigthe type of event that will trigger the interrupt (rising,falling,both)
